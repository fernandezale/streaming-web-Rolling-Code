# STREAMING WEB - #1 PROYECTO ROLLINGCODE
En este proyecto buscamos crear una página de streaming para poner en práctica los conocimientos del módulo 1 - "Creación y diseño de páginas web".

Tecnologias utilizadas:
- HTML
- CSS
- BOOTSTRAP
- GIT
- GITHUB

Nos inspiramos en páginas como KICK y NETFLIX. Elegimos un tema oscuro y, como color principal, uno de la gama del naranja.
El logo y el nombre de la página fueron creados por IA y sometidos a votación de todos los miembros del equipo.
