# STREAMING WEB - #1 PROYECTO ROLLINGCODE
En este proyecto buscamos crear una página de streaming para poner en práctica los conocimientos del módulo 1 - "Creación y diseño de páginas web".
Tecnologias utilizadas:
- HTML
- CSS
- BOOTSTRAP
- GIT
- GITHUB

Nos inspiramos en paginas como KICK y NETFLIX. Elegimos un tema oscuro y como color principal uno de la gama del naranja.
El logo y el nombre de la pagina fue creado por IA y sumetido a votacion por todos los miembros del equipo.

Queremos dar las gracias a RollingCode, Pablo Marino y Mateo Yapur por ser parte de nuestro aprendizaje y crecimiento.
